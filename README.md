# MEXC Micro Maker Bot v0057 Freeze/Version Guard Fix

Исправления v0057:
- защита от двух одновременно запущенных процессов на одном сервере (`/tmp/mexc_micro_maker_bot.lock`);
- `run_polling(drop_pending_updates=True)`, чтобы после редеплоя не обрабатывать старые callback/update;
- все edit/send Telegram панели обёрнуты timeout, чтобы UI-lock не зависал навсегда;
- `/log_full` больше не удаляет команду молча: сначала пишет “готовлю лог”, потом отправляет файл;
- добавлены `/doctor` и `/panel_reset` для диагностики и восстановления панели;
- версия обновлена на v0057 / `wave_price_tsunami_v0057`.

Важно при деплое: запустить только один процесс/контейнер бота. Если Telegram показывает разные версии, старый процесс всё ещё жив.
